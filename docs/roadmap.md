# 今後の拡張

## 優先度高

- Firestore保存
- Firebase Authentication
- OpenRouterモデル一覧取得
- URLインポート後のAI要約
- 原稿差分を見ながらリテイクを採用するUI
- 話数の並び替え
- 作品名、話タイトルの一覧性改善

## 優先度中

- チャットログ検索
- ボード変更履歴
- 本稿の章分割
- 生成結果の採用、却下、保留
- 本文の手動編集
- Markdownまたは縦書きプレビュー

## 優先度低

- 公開機能
- 読者向けリーダー
- ユーザー間共有
- 課金
- チーム執筆

## 技術的課題

- LocalStorage容量制限
- OpenRouterモデルごとのJSON出力安定性
- ボード自動更新の過剰反映
- 長期チャットログのコンテキスト圧縮
- 作品全体と話数単位の文脈配分
