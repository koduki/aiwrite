# サービス仕様

## 概要

アイライトは、ユーザー自身のOpenRouter APIキーを使い、AI作家と対話しながら小説を作るBYOK型の創作支援アプリである。

MVPでは公開、課金、ユーザー間交流は扱わない。主目的は、ユーザーがAIと会話しながら複数作品・複数話の構想、試し書き、調整、本稿執筆を進めることに置く。

## コンセプト

ユーザーは監督兼プロデューサー、AIは編集者、サンプル脚本家、共著者、ゴーストライターとして振る舞う。

AIにただ本文を書かせるのではなく、会話のラリーを通じて作品設定を育て、創作ボードへ反映し、固まった設定から本稿を書き進める。

## 対象ユーザー

- 小説を書きたいが、設定整理や初稿作成に負荷を感じる人
- AIとの壁打ちで作品の方向性を固めたい人
- プロット、キャラクター、世界観、文体を試しながら決めたい人
- 複数作品や複数話を並行して扱いたい人

## MVPの範囲

- OpenRouter BYOK
- ChatGPT風のスマホ中心UI
- 複数作品管理
- 作品ごとの複数話管理
- 作品単位の創作ボード
- 話数単位のチャットログと本文
- 4フェーズ制の対話
- 会話に応じた創作ボード自動更新
- URLまたはテキストからの素材インポート
- Cloud Run向けDockerfile

## MVPで扱わないもの

- 公開機能
- 課金
- ユーザー間フォロー、レビュー、ランキング
- サーバー側永続化
- Firebase Authenticationの本格導入
- Firestore保存
- OpenRouterキーのサーバー保存
